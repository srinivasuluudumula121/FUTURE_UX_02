# Information Architecture

## App Structure

The Bhai Looks Saloon appointment booking app was designed with a mobile-first structure focused on reducing booking friction.

The application architecture prioritizes:

- Fast booking flow
- Minimal navigation complexity
- Clear decision-making
- Service-first interaction

## Main Navigation

The app uses a simple bottom navigation system with:

### Home
Browse services and start booking.

### Appointments
View upcoming and completed bookings.

### Profile
Manage personal details and booking history.

## Booking Structure

Home
→ Select Service
→ Choose Barber
→ Select Time Slot
→ Phone Verification
→ Review Booking
→ Booking Confirmation

## Design Goal

The architecture was intentionally simplified to reduce confusion for first-time users and make appointment booking quick and intuitive.
