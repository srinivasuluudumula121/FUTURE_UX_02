# Design System

A minimal and premium mobile-first design system was created to maintain visual consistency throughout the Bhai Looks Saloon appointment booking app.

The design system was intentionally kept simple to prioritize usability and reduce distractions during the booking process.

## Design Style

The visual direction focuses on:

- Premium minimal aesthetic
- Masculine grooming feel
- Warm and professional appearance
- Mobile-first usability
- Clear interaction hierarchy

## Color System

The color palette uses a warm neutral background combined with premium accent colors.

### Primary Accent Color
Muted orange was used to indicate:

- Primary call-to-action buttons
- Selected states
- Active navigation
- Important booking actions

### Neutral Background
Soft warm beige tones were used to create a premium and calm atmosphere.

### Success Color
Muted green was used for:

- Booking success states
- Confirmation indicators
- Positive appointment status

## Typography

Typography was designed for readability on mobile screens.

The system prioritizes:

- Clear hierarchy
- Strong readability
- Consistent type scaling
- Easy scanning

Typography levels include:

- Screen headings
- Section titles
- Card labels
- Supporting text
- Small captions

## Components

Reusable components were designed for consistency across screens.

Components include:

- Buttons
- Service cards
- Barber cards
- Time slot chips
- Input fields
- Navigation bar
- Booking summary cards

## Spacing & Radius

Consistent spacing and corner radius were applied to maintain a clean and premium interface across the application.
