# Bhai Looks Saloon – Mobile Appointment Booking App

UI/UX Design Project for Future Interns – Task 02

---

## Project Overview

This project is a mobile-first appointment booking application designed for **Bhai Looks Saloon**, a real local salon in Pamuru.

The goal was to redesign the traditional manual appointment process and create a professional mobile experience that reduces booking friction, minimizes waiting time, and improves customer convenience.

The application was designed with a strong focus on:

- Simple booking experience
- Mobile-first usability
- Clear navigation
- Reduced user friction
- Professional visual consistency

---

## Problem Statement

Many local salons still manage appointments using:

- Walk-ins
- Phone calls
- WhatsApp communication

This creates several issues:

- Missed appointments
- Scheduling confusion
- Long waiting times
- Poor customer experience
- Manual booking inefficiencies

This project solves these challenges through a structured digital appointment booking system.

---

## Target Users

### Returning Customers
Users who already know their preferred barber and want quick appointment booking.

### First-Time Customers
Users unfamiliar with salon staff who require barber recommendations and booking guidance.
